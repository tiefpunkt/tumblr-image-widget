# Tumblr Image Widget
Displays slideshow of images in a tumblr log.

* Uses Tumblr API v1 http://www.tumblr.com/docs/en/api/v1

## Warning!
Stil pretty ugly, work in progress. Feel free to contribute.

## Libraries in use
* jQuery
* jQuery Image Scale http://imgscale.kjmeath.com/
